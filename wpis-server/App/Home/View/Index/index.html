QWadmin

基于thinkphp与aceadmin模板的后台管理系统

===============================安装说明================================

1.sql.sql文件导入到数据库中。

2.在文件App/Common/Conf/db.php设置数据库相关信息

3.在文件App/Common/Conf/config.php设置COOKIE_SALT。

4.配置完成。

后台地址：进入后台管理系统 账号admin 密码 admin
