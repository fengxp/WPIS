openuploadify
==========

在吕大豹的Huploadify 模仿的uploadify的基础上增加如下功能：
1. 增加动态获取网页参数
2. 增加php删除页面，可删除已经上传文件。


以下为原Huploadify的功能
jQuery文件上传插件，HTML5版uploadify，保持与uploadify一致的API，完全山寨。Uploadify官网：[http://www.uploadify.com/](http://www.uploadify.com/)
